# Frontend Mentor - Blog Preview Card Solution

This is my solution to the Blog Preview Card challenge on Frontend Mentor. The goal of this project was to build a clean and responsive blog card layout that closely matches the provided design using HTML and CSS.

## What I Learned

In this project, I practiced building a card component with a clear content hierarchy and clean spacing. I improved my understanding of Flexbox by using it to align the author image and name properly, and I also learned how to control responsiveness by combining `width`, `max-width`, and page padding.

I also became more comfortable using semantic HTML elements such as headings and spans for structured content, and I improved my attention to detail when matching a design more closely.

## Challenges I Faced

One of the main challenges was getting the spacing and typography to look close to the original design. I had to adjust font sizes, padding, and margins several times to make the card feel balanced.

Another challenge was aligning the author image and name at the bottom of the card. I solved this by using Flexbox with `display: flex`, `align-items: center`, and `gap` to place them correctly in one row.

## What I’m Proud Of

I’m most proud of creating a clean final layout that looks very close to the design and works well on smaller screens. I also feel more confident now in using Flexbox for alignment and in deploying my project with GitHub Pages.

## What I Would Do Differently Next Time

Next time, I would focus even more on writing cleaner and more reusable CSS from the beginning. I would also pay closer attention to typography and spacing earlier in the project so I can reduce the number of adjustments later.
